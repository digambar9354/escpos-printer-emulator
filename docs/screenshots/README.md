# Screenshots

Drop the app screenshots used by the main `README.md` into this folder, using these exact file names so the links resolve:

| File name | What to capture |
|-----------|-----------------|
| `receipt-preview.png` | The **Receipt** tab showing a rendered receipt (dark theme). |
| `command-log.png` | The **Commands** tab showing the colour-coded command log. |
| `settings.png` | The **Settings** tab (optional). |

## How to capture (Windows)
1. Run the app: `cargo run --release`
2. Press **Win + Shift + S** to open the Snipping Tool, select the window area.
3. Paste into Paint (or any editor) and **Save as PNG** into this folder with the name above.

Keep images reasonably sized (e.g. ≤ 1600px wide) so the repo stays light.
